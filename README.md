# study_terasoluna

bcmアプリを理解できるだけの素養が無いので
写経などで少し基礎勉強を追加する

terasoluna使うとややこしいからそれぞれSpringFrameworkを直に使ったバージョンで  
チュートリアルこなす

## 共通環境

- Gradle
- IntelliJ
- Java17
